# 云端控件自测

更新日期：2026-07-13

## 测试方法

运行 `pnpm test:cloud-controls`。测试启动生产构建，通过 Electron `sendInputEvent` 发送真实的鼠标移动、按下和松开事件，不使用 DOM `click()`。

## 覆盖结果

| 控件 | 验证内容 | 结果 |
| --- | --- | --- |
| 主界面“云端” | 点击后显示云端弹窗 | 通过 |
| 右上角关闭 | 点击后移除弹窗 | 通过 |
| 重新建立云端空间 | 调用 IPC 并更新账号凭证输入框 | 通过 |
| 复制密钥 | 写入系统剪贴板，测试后恢复原内容 | 通过 |
| 打开网页看板 | 调用打开网页 IPC | 通过 |
| 生成配对码 | 保存当前空间并生成一次性配对码 | 通过 |
| 加入同步空间 | 兑换独立设备凭据、保存并立即同步 | 通过 |
| 自动同步 | 点击后切换开启状态 | 通过 |
| 取消 | 点击后关闭弹窗 | 通过 |
| 保存并同步 | 保存设置、触发同步并在成功后关闭弹窗 | 通过 |

## 回归约束

- 主界面默认必须为 `no-drag`，仅顶部标题栏允许拖动窗口。
- 云端弹窗和全部后代控件必须为 `no-drag`。
- 保存成功后必须提供可见反馈并关闭弹窗。
