CodexMeter 硬件刷机工具

给 ESP32-C3 小屏刷入 CodexMeter 固件

无需安装 Arduino IDE，使用 Chrome 或 Edge 打开本页，连接 ESP32-C3，点击安装即可写入小屏固件。 刷入后设备会提供蓝牙连接、HTTP 推送和 Wi-Fi 配网页。

蓝牙直连 桌面端可直接搜索并连接 CodexMeter 小屏。
网络推送 配网后通过局域网 HTTP 自动同步额度状态。
网页配网 刷机完成后连接 CodexMeter-Setup 热点配置 Wi-Fi。

开始刷机

按顺序操作，刷机过程会清空 ESP32-C3 上原有程序。

  1. 1 用 USB 数据线连接 ESP32-C3。
  2. 2 点击下方安装按钮，选择 ESP32-C3 对应串口。
  3. 3 确认安装，等待固件写入完成。
  4. 4 连接 CodexMeter-Setup 热点，打开 192.168.4.1 配网。

固件版本：v0.1.0

如果浏览器看不到设备，请按住开发板 BOOT 键后重新插入 USB，或更换支持数据传输的 USB 线。 安装按钮和系统串口授权弹窗由浏览器组件提供，个别类型可能显示英文。
