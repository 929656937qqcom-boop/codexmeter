CodexMeter Agent for macOS

1. 在一台已连接云端的 CodexMeter 电脑上打开“云端”，点击“生成”获取 8 位配对码。
2. Mac 上双击“Install CodexMeter Agent.command”，输入配对码。
3. 安装完成后，采集器每 5 分钟自动读取 ~/.codex 并同步匿名 Token 汇总。
4. 双击“Open Dashboard.command”可打开云端看板。
5. 不再使用时，双击“Uninstall CodexMeter Agent.command”。

隐私：不会上传提示词、回复、项目路径、文件内容、Codex OAuth 或原始线程 ID。

若 macOS 阻止首次运行：右键文件，选择“打开”，再确认一次。
